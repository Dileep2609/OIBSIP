# 📖 About The Project

This Todo Web Application helps users manage daily tasks efficiently.

Users can:

- Add new tasks
- Edit existing tasks
- Delete completed tasks
- Search tasks instantly
- Filter completed and pending tasks
- Store tasks permanently using Local Storage

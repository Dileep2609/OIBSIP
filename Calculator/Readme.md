# OIBSIP - Scientific Calculator

This repository contains the Level 2 Task 1 project completed as part of the Oasis Infobyte Web Development and Designing Internship.

## 🚀 Project Name

Scientific Calculator

## 📌 Description

A modern Scientific Calculator built using HTML, CSS, and JavaScript.  
The calculator supports basic arithmetic operations along with scientific functions and advanced UI features.

## ✨ Features

- Basic Arithmetic Operations

  - Addition
  - Subtraction
  - Multiplication
  - Division

- Scientific Functions

  - Square Root
  - Power
  - Sin
  - Cos
  - Tan
  - Logarithm
  - Pi Value

- Additional Features
  - Responsive Design
  - Keyboard Support
  - Dark/Light Mode
  - Sound Effects
  - Calculation History
  - Button Animations
  - Mobile Friendly UI

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

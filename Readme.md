# Oasis Infobyte Web Development Internship

This repository contains all the projects completed as part of the Oasis Infobyte Web Development and Designing Internship.

---

# 🚀 Projects Included

## 🔢 1. Scientific Calculator

A modern scientific calculator with advanced functionalities and responsive UI.

### Features

- Basic Arithmetic Operations
- Scientific Functions
- Dark / Light Mode
- Keyboard Support
- Responsive Design
- Calculation History
- Sound Effects
- Mobile Friendly

---

## 🏆 2. Tribute Page - Dr. A.P.J Abdul Kalam

A beautiful tribute webpage dedicated to Dr. A.P.J Abdul Kalam.

### Features

- Responsive Design
- Biography Section
- Awards & Honors
- Image Gallery
- Inspirational Quotes
- Timeline Section
- Smooth Animations
- Modern UI

---

## ✅ 3. Advanced Todo Web App

A responsive Todo application with task management features.

### Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Completed
- Search Tasks
- Filter Tasks
- Local Storage Support
- Dark / Light Mode
- Responsive Design

---

## 🔐 4. Login Authentication System

A modern authentication system using Local Storage.

### Features

- User Registration
- User Login
- Logout System
- Show / Hide Password
- Form Validation
- Responsive UI
- Dark / Light Mode
- Local Storage Authentication

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API
